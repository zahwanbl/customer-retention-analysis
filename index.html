Key Findings

Insights & Recommendations

Key business insights derived from customer behavior analysis and recommended actions to improve retention and revenue.

📊 Key Insights

  • • The majority of customers (97%) are one-time buyers, indicating very low retention.
  • • Repeat customers represent only a small portion of the total customer base (~2.87%).
  • • A group of high-value customers exists, but many do not return after their first purchase.
  • • Very few customers consistently generate high revenue (high-value repeat segment).

💡 Recommendations

  • • Target high-value one-time customers with personalized retention campaigns.
  • • Implement loyalty programs to encourage repeat purchases.
  • • Improve post-purchase engagement to increase customer return rate.
  • • Focus on converting occasional buyers into long-term valuable customers.

Project Details

Dataset & Methodology

This section explains the dataset used and the analytical steps taken to generate customer retention insights.

📂 Dataset

The analysis uses the Brazilian E-Commerce Public Dataset by Olist, which contains order, customer, and payment information from an online marketplace.

Tables Used

  • • olist_customers_dataset
  • • olist_orders_dataset
  • • olist_order_payments_dataset

🛠️ Methodology

  • • Cleaned and prepared the datasets by checking missing values and formatting date columns.
  • • Merged customer, order, and payment data using customer_id and order_id.
  • • Aggregated data into customer-level metrics such as total orders, total spending, first purchase, and last purchase.
  • • Classified customers into one-time and repeat buyers.
  • • Applied RFM-based segmentation to identify customer value and retention opportunities.

Case Study

STAR Method

A structured summary of the project using Situation, Task, Action, and Result.

Situation

An e-commerce dataset showed that most customers made only one purchase, raising concerns about low customer retention and long-term revenue sustainability.

Task

I aimed to analyze customer transaction data to identify customer value patterns, repeat purchase behavior, and revenue contribution by customer segment.

Action

  • • Cleaned and prepared customer, order, and payment datasets.
  • • Used Python to merge tables and aggregate customer-level metrics.
  • • Calculated total orders, total spending, recency, frequency, and monetary value.
  • • Performed RFM analysis to segment customers.

Result

  • • Found that 97.13% of customers were one-time buyers.
  • • Identified that only 2.87% of customers made repeat purchases.
  • • Found high-value one-time customers as a key retention opportunity.
  • • Recommended targeted retention campaigns and loyalty strategies.
